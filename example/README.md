# flutter_book_reader_example

Example app for the [`flutter_book_reader`](../) package: a bookshelf plus the
reader, with sample books served from `assets/books.json`.

## Run

```bash
flutter pub get
flutter run
```

## What it shows

- Implementing `BookSource` against a local JSON asset
  (`lib/data/asset_json_book_source.dart`).
- Sharing an `InMemoryReaderProgressStore` across books to remember positions.
- Opening `BookReader` from a list of books.

## Regenerating sample data

Book content lives in `assets/books.json`, generated by:

```bash
dart run tool/gen_books.dart
```
